## Manual Setup

`git clone https://github.com/jaydanurwin/anoun-gatsby-website.git`

### Install Node Modules & CLI's

```bash
# if you don't have gatsby-cli already
npm install -g gatsby-cli
# install modules
npm install
```

### Develop Locally

```bash
npm run start

# or

gatsby develop
```
